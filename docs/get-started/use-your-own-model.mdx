# Use Your Own Model

If you want to use your own model in OOMOL Studio for block authoring and block orchestration, follow the setup tutorial below.

:::info
OOMOL Studio itself is free to use. After you configure your own model, AI-assisted authoring requests go through your own model provider instead of consuming OOMOL Credits.

If you already have available model quota, you can start using Studio without extra model spending on OOMOL. Model usage is still billed by your provider.
:::

## After Setup

- Use your own model quota to write Blocks and compose workflows in Studio.
- Keep Studio-side local development free to start.
- Avoid spending OOMOL Credits for this part if you already have your own model access.
