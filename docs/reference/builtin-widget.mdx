# Builtin Widget
In order to allow users to quickly write the UI of the block, we map the schema field of Options to the built-in widget.
Here we show how to declare the use of built-in widgets in a yaml file and show how the widgets look in the graph.

For complete configuration file, refer to [Block](http://localhost:3000/docs/reference/block#properties).
Here we only discuss the value of the `options` field. such as:
```
...
options:
  userOptionKey:
    {schemas below}
...
```

## Boolean
**yaml**
```
schema:
  type: boolean
```

**yaml**
```
schema:
  type: integer

## Select
**yaml**

:::caution
You should make items in `enum` field be unique.
:::
```
schema:
  type: string # type of Select Widget is unrestricted.
  enum: ["csv", "xls", "txt"]
```
or:

:::caution
You should make `value` in `metaOptions` field be unique.
:::
```
schema:
  type: string  # type of Select Widget is unrestricted.
  "ui:widget": select
  "ui:options":
    metaOptions: 
      - label: csvLabel
        value: csv
      - label: xlsLabel
        value: xls
```
**view in graph**

<img className={styles["sample-image"]} src={selectSample}/>

## Multi Select
**yaml**

:::caution
You should make `value` in `metaOptions` field be unique.
:::

```
schema:
  type: array
  "ui:widget": multiSelect
  "ui:options":
    metaOptions: 
      - label: csv
        value: csv
      - label: xls
        value: xls
      - label: txt
        value: txt

## Object
**yaml**
```
schema:
  type: object
  required:
    - stringInput
    - colorPicker
  properties:
    stringInput:
      type: string
    colorPicker:
      type: string
      "ui:widget": color
    objectInside:
      type: object
      properties:
        input:
          type: string
```

## Any Of
Check: [AnyOf](https://json-schema.org/understanding-json-schema/reference/combining#anyOf)

**yaml**
```
schema:
  anyOf:
    - type: object
      properties:
        A:
          type: string
        B:
          type: number
    - type: number
    - type: string
```

## All Of
Means all of schema conditions must be satisfied. Check: [AllOf](https://json-schema.org/understanding-json-schema/reference/combining#allof)
