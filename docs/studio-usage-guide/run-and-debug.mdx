## 1. Install

### 1.1 Bootstrap Install Dependencies

> The `package.oo.yaml` file defines startup scripts to automate the bootstrap installation of project dependencies:
>
> 1. Node.js dependencies will be installed based on the package.json file
> 2. Python dependencies will be installed based on the requirements.txt file
> 3. System-level dependencies need to be manually added to the bootstrap script via apt-get

<img src={bootstrap} width="720" />

### 1.2 Developers Install Package

> 1. Installation through interface: Python packages will be automatically added to the requirements.txt file, Node.js packages will be automatically added to the package.json file
> 2. Installation through command line: Python packages need to be manually added to the requirements.txt file, while Node.js packages will automatically update package.json
