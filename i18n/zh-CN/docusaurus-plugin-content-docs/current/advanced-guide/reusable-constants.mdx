# 可复用的常量值

你可以在系统区块包中找到值区块。

值区块是 OOMOL Studio 内置的一个特殊的共享区块，设计初衷是为用户提供一个可以复用的常量输入。

这里值区块的作用等价于为两个 Python 区块的 input 输入填入同一个值。
