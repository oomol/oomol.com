# 可复用的常量值

你可以在系统区块包中找到值区块。

值区块是 OOMOL Studio 内置的一个特殊的共享区块，设计初衷是为用户提供一个可以复用的常量输入。

这里值区块的作用等价于为两个 Python 区块的 input 输入填入同一个值。

值区块本身不属于一个可以执行的区块，因此它的输出内容不会触发下游节点的运行。

对于流来说，值区块节点也不会成为流的开始节点，它仅仅有一个赋值的功能。

因此我们使用了特殊的连线来表示这个机制。
