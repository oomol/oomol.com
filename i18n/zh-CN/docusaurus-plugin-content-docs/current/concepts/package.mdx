# 包

包是项目做为一个完整产品发布后的名称。

当你开发的流或者区块实现的功能可以被其他项目或者用户复用的时候，你可以选择将这些内容发布为一个包。

包是项目做为一个完整产品发布后的名称。它内涵了一个或者多个流和区块。包可以单独使用其中的某一项内容，但是下载必须是全部包的内容。

你可以在商店页面找到不同功能的包：

## 包的类型

包有两种发布方式，对应了不同的使用范围。

如果你自己做的包仅满足了自己的使用需要，而不想共享给别人别人时，你可以选择发布到本地。

如果发布到云端，那么所有 OOMOL 用户都可以看到并下载这个包：
