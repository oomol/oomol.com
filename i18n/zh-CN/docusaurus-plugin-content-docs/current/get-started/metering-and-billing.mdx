# 计量与计费

OOMOL Studio 内置了 LLM 区块。你可以直接选择 OOMOL 接入的大语言模型进行调用，不需要自己再写接入代码。

OOMOL Studio 应用本身可以免费使用，但调用 LLM 区块时会产生服务费用。为了统一不同渠道的结算方式，我们会把这部分费用折算成 OOMOL 点数（Credits）扣除。

:::info
如果你想使用自己的大语言模型 API Token，可以直接看[配置自己的大模型](/zh-CN/docs/get-started/use-your-own-model)。

配置完成后，Studio 里的这部分 AI 编写与编排会直接走你自己的模型额度，不再消耗 OOMOL 点数。
:::

## OOMOL 点数（Credits）

OOMOL Studio 接入了多种大语言模型。由于不同渠道的结算方式不一样，这些 AI 服务产生的费用会统一折算成 OOMOL 点数。

每个月 OOMOL 都会赠送一定的点数额度。你可以登录 [OOMOL 控制台](https://console.oomol.com) 查看点数使用情况和余额。

你也可以在 [价格页面](/zh-CN/pricing) 查看点数价格。

如果有需要，也可以在 OOMOL 控制台购买额外点数。订阅方式不会影响额外点数的充值价格。
