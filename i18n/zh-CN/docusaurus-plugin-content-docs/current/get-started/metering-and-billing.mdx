# 计量计费

OOMOL Studio 内置了 LLM 区块，可以让用户选择 OOMOL 接入的各种大语言模型并进行交互，用户不用编写接入代码。

OOMOL Studio 应用本身是免费使用的，但是在调用 LLM 区块时会产生相应的服务费用，这些费用我们会折算为 OOMOL 积分进行扣除。

:::info
如果你想使用自己的大语言模型 API Token，你可以选择通过小脚本区块编写代码来接入模型。
:::

## OOMOL 积分

OOMOL Studio 接入了目前比较受欢迎的各种大语言模型，同时考虑到用户使用成本也选择了不同的渠道，考虑到每个渠道的结算方式不同，我们将使用这些 AI 服务产生的费用统一结算为 OOMOL 积分。

每个月 OOMOL 都会赠送给用户一定的积分额度，用户可以在 [Console 平台](https://console.oomol.com)登录后查看积分的使用情况和余额。

也可以在[价格页面](/zh-CN/pricing)查看积分价格。

如果有需要，你可以在 Console 平台购买额外的积分，或者改变订阅方式获得积分的购买折扣。
