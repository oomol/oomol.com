# 配置自己的大模型

如果你想在 OOMOL Studio 里使用自己的模型来写 Block、编排 Block，可以直接跟着下面这个视频完成配置。

:::info
OOMOL Studio 本身可以免费使用。配置成你自己的模型后，这部分 AI 调用会走你自己的模型服务商额度，不再消耗 OOMOL 积分。

如果你本来就有现成的模型额度，基本可以直接开始用 Studio；模型费用仍由你的服务商结算。
:::

## 配好后可以做什么

- 用自己的模型在 Studio 里写 Block、改 Block、编排 Block。
- 本地开发阶段不需要额外消耗 OOMOL 积分。
- 如果你已经有现成的模型账号和额度，起步会更轻。
