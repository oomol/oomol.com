# 合并为一个可复用的区块

在上一节中，我们基于加法器创建了一个幂运算器，创建了一个计算直角三角形斜边长度的工作流。现在我们希望将这个工作流转换为一个可复用的区块，以便在其他工作流中使用。否则每次都需要重新创建这个工作流，显然不太方便。
