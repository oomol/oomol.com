# 概述

## 1. 核心特点

OOMOL 是一个 AI 驱动的工作流 IDE，专为数据分析、自动化脚本开发以及智能体构建设计。相比同类产品，我们更专注于开源生态和可编程能力。

### 1.1. 开源

我们采用的商业模式是：底层执行引擎完全开源，上层应用作为增值服务提供便利性。

- 底层执行引擎开源，便于开发者理解运行机制并进行功能定制
- 支持导出为 Docker 镜像，实现跨平台无缝部署（开发中）

### 1.2. 可编程

我们的核心是 Scriptlet Block，支持开发者使用 Python 和 Node.js 实现自定义业务逻辑。

- 传统工作流产品通常限制用户使用预设模块进行拼接，导致业务实现受限
- 我们将工作流与开源生态无缝集成，允许直接在工作流中使用各类开源库，赋予产品无限可能性

### 1.3. AI 原生

- 集成 AI 辅助编程功能，降低代码编写门槛
- 支持基于大模型 function calling 的自动化编程
- 内置多种 AI 大模型，开发者可直接选用而无需复杂配置
