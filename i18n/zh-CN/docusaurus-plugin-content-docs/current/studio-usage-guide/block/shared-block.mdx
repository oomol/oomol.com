# 共享区块

`共享区块`是从`小脚本区块`转换而来的，可以重用的区块。

当一个`小脚本区块`的功能具有通用性，开发者想要将其抽象成一个工具重复使用时，就可以将`小脚本区块`转换成一个`共享区块`。

## 生成

你可以通过`小脚本区块`的右上角的按钮转化成`共享区块`。

## 编辑代码

从`小脚本区块`转换来的`共享区块`会保留原来的设置和代码，只是代码会隐藏起来。

你可以通过以下两种区块菜单查看并编辑代码。
