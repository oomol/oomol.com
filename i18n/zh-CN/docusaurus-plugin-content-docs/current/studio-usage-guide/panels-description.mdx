# 📝 面板说明

介绍重要面板的功能和用途。

<img src={panel} width="720" />

## 1. 🟡 顶部面板

该面板主要用于运行工作流。

| 模块               | 描述               |
| ------------------ | ------------------ |
| 运行流             | 运行当前工作流程   |
| 重新运行节点       | 运行选定的节点     |
| 在文本编辑器中打开 | 看工作流的yaml源码 |

## 2. 🔴 左面板

该面板主要用于创建配置工作流和块。

| 模块   | 描述                                                                                                 |
| ------ | ---------------------------------------------------------------------------------------------------- |
| Flows  | 工作流画布，一个项目可以创建多个。同一个项目中的所有流依赖项都可以共享。                             |
| Blocks | 产品的核心模块，包含 Scriptlet Block 和 Shared Block 两种。 Shared Block 由 Scriptlet Block 转化而来 |
| 工作区 | 源代码。                                                                                             |
| 存储   | 存储工作流中需要使用的静态资源的位置。静态资源包括图像、视频和表格内容。                             |
| 密钥   | 工作流中的某些块需要调用第三方服务。可以使用Secrets模块访问调用服务所需的机密。                      |

## 3. 🔵 底部面板

该面板主要用于工程师的深度开发和调试工作流程。

| 模块         | 描述                                                                                                                                                 |
| ------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| 终端         | 连接到容器 OVM 的终端。需要注意的是，在 OVM 中运行的系统 Linux。                                                                                     |
| Packages     | 用于安装 Python 和 JS 的第三方库的面板。需要注意的是，通过面板安装的 Python 库会自动添加到 `requirements.txt` 中。直接使用终端安装的用户需要手动添加 |
| 工作流人日志 | 工作流运行期间生成的日志。                                                                                                                           |
| 容器脚本     | 在 OVM 容器中首次启动项目时需要自动执行的脚本。                                                                                                      |
| 问题         | 编码错误                                                                                                                                             |

## 4. ⚪ 右面板

该面板主要用于参考现成的积木。

面板上方为封装，下方为封装中的功能模块，需要注意的是，如果没有选择，则会列出所有的封装和功能模块。

| 模块          | 描述                                           |
| ------------- | ---------------------------------------------- |
| 内建 Packages | OOMOL 会预装一些必要的软件包                   |
| Package 商店  | 您可以在 Packages Store 中下载其他用户共享的包 |
