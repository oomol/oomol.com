# 📝 面板描述

介绍重要面板的功能和用途。

<img src={panel} width="720" />

## 🟡 顶部面板

此面板主要用于运行工作流。

| 模块               | 描述                     |
| ------------------ | ------------------------ |
| 运行工作流         | 运行当前工作流           |
| 重新运行节点       | 运行选定的节点           |
| 在文本编辑器中打开 | 查看工作流的 YAML 源代码 |

## 🔴 左侧面板

此面板主要用于创建配置工作流和模块。

| 模块   | 描述                                                                           |
| ------ | ------------------------------------------------------------------------------ |
| 工作流 | 工作流画布，一个项目可以创建多个工作流。项目中的所有工作流依赖项可以共享。     |
| 模块   | 从 Scriptlet 转换而来的模块，是一个可重用的自定义工具                          |
| 工作区 | 源代码。                                                                       |
| 存储   | 存储工作流中需要使用的静态资源的位置。静态资源包括图像、视频和表格内容。       |
| 密钥   | 工作流中的某些模块需要调用第三方服务。调用服务所需的密钥可以通过密钥模块访问。 |

## 🔵 底部面板

此面板主要用于工程师深入开发和调试工作流。

| 模块       | 描述                                                                                                                                                 |
| ---------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| 终端       | 连接到容器 OVM 的终端。需要注意的是，OVM 中运行的系统是 Linux。                                                                                      |
| 包管理     | 用于安装 Python 和 JS 的第三方库的面板。需要注意的是，通过面板安装的 Python 库会自动添加到 `requirements.txt` 中。直接使用终端安装的用户需要手动添加 |
| 工作流日志 | 工作流运行期间生成的日志。                                                                                                                           |
| 容器脚本   | 在 OVM 容器中首次启动项目时需要自动执行的脚本。                                                                                                      |
| 问题       | 代码错误                                                                                                                                             |

## ⚪ 右侧面板

此面板主要用于引用现成的模块。

面板的上半部分是包，下半部分是包中的功能模块。需要注意的是，如果没有选择任何内容，将列出所有包和功能模块。

| 模块   | 描述                                 |
| ------ | ------------------------------------ |
| 内置包 | OOMOL 将预安装一些必要的包           |
| 包商店 | 您可以在包商店中下载其他用户共享的包 |
