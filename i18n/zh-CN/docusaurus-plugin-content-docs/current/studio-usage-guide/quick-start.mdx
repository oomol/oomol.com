# 🚀 快速开始

## 1. 下载安装

- 1.1 从[下载页面](https://oomol.com/zh-CN/downloads)下载最新版本的 `OOMOL Studio`。
- 1.2 Mac 可以直接安装使用。
- 1.3 Windows 需要跟随安装指南启动 [WSL2](https://www.youtube.com/watch?v=MrZolfGm8Zk) 并重新启动才能运行。
