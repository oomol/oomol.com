# 🚀 快速开始

按照文档从下载到启动您的第一个工作流程。

## 1. 下载

从[登陆页面](https://oomol.com)下载最新版本的 `OOMOL STUDIO`

## 2. 安装

1. Mac 可以直接安装使用。
2. Windows 需要按照安装指南启动 [WSL2](https://www.youtube.com/watch?v=MrZolfGm8Zk) 并重新启动才能运行。

## 3. 通过Github登录

我们希望用户将工作流源代码推送到 GitHub 上，所以我们首先支持 GitHub 登录。

## 4. 从社区启动

您可以在社区中找到您需要的工作流程，然后单击按钮使用 `OOMOL STUDIO` 打开.
