# 🚀 快速开始

按照文档从下载到启动您的第一个工作流。

## 1. 下载

从[登录页面](https://oomol.com)下载最新版本的 `OOMOL STUDIO`；

## 2. 安装

1. Mac 可以直接安装和使用。
2. Windows 需要按照安装指南启动 [wsl2](https://www.youtube.com/watch?v=MrZolfGm8Zk) 并重启后才能运行。

## 3. 使用 Github 登录

我们希望用户将工作流源代码推送到 github 上，所以我们首先支持 github 登录。

## 4. 从社区启动

您可以在社区中找到所需的工作流，并点击按钮通过 `OOMOL STUDIO` 打开。
