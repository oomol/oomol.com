## 1. 安装

### 1.1 引导安装依赖项

> `package.oo.yaml` 文件定义了启动脚本，用于自动化引导安装项目依赖：
>
> 1. Node.js 依赖项将基于 package.json 文件进行安装
> 2. Python 依赖项将基于 requirements.txt 文件进行安装
> 3. 系统级依赖需要通过 apt-get 手动添加到启动脚本中

<img src={bootstrap} width="720" />

### 1.2 开发依赖包安装

> 1. 通过界面安装：Python 包会自动添加到 requirements.txt 文件，Node.js 包会自动添加到 package.json 文件
> 2. 通过命令行安装：Python 包需要手动添加到 requirements.txt 文件，Node.js 包则会自动更新 package.json

<img src={package_install} width="720" />
