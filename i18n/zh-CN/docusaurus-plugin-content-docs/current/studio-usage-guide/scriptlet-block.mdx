# 💡 小脚本 & 共享区块

## 1. 小脚本

Scriptlet Block 是一个核心的基本模块。OOMOL 中的所有块内容都是从 Scriptlet Block 转换而来的。

当内置 Block 不能满足创建工作流的需要时，用户可以基于 Scriptlet Block 编程进行补充。

### 1.1 使用 Scriptlet Block

> 您可以从左侧的Scriptlet Block工具栏中拖出并使用它。

<img src={scriptlet} width="720" />

### 1.2 配置 Scriptlet Block

> 添加或删除输入输出参数，并定义类型。

<img src={scriptlet_config} width="720" />

### 1.3 Scriptlet Block 代码

> 支持使用Nodejs和Python处理业务逻辑。支持社区开源库引用，支持完成和AI辅助编程。

<img src={scriptlet_code} width="720" />

### 1.4 Scriptlet Block 预览

> 这个功能是帮助开发者预览数据，我们支持几乎所有常用数据的预览。

<img src={scriptlet_preview} width="720" />

## 2. 共享区块

Shared Block 是从 Scriptlet Block 转换而来的，也是一个重要的基础模块，当一个 Scriptlet Block 的功能具有通用性，开发者想要将其抽象成一个工具重复使用时，就可以将 Scriptlet Block 转换成一个 Shared Block。

### 2.1 生成 Shared Block

> 您可以通过 Scriptlet Block 的右上角的按钮转化成 Shared Block。

<img src={scriptlet_to_block} width="720" />

> 也可以在左侧工具栏创建 Shared Block。

<img src={create_block} width="720" />

### 2.2 使用 Shared Block

> 您可以从左侧的共享块工具栏中拖出并使用它。

<img src={block} width="720" />

### 2.3 配置 Shared Block

> 您可以单击共享块来配置输入和输出参数。
> 这里的交互与 Scriptlet Block 相同。

<img src={block_edit} width="720" />

### 2.4 Shared Block 代码

> 您可以单击共享块的代码图标以更改其代码

<img src={block_code} width="720" />
