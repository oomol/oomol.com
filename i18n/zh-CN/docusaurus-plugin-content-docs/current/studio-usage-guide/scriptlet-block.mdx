# 💡 Scriptlet Block

Scriptlet Block 是一个核心的基本模块。oomol 中的所有块内容都是从 Scriptlet Block 转换而来的。

当内置 Block 不能满足创建工作流的需要时，用户可以基于 Scriptlet Block 编程进行补充。

## 1. 使用 Scriptlet Block

> 您可以从左侧的Scriptlet Block工具栏中拖出并使用它。

<img src={scriptlet} width="720" />

## 2.配置 Scriptlet Block

> 添加或删除输入输出参数，并定义类型。

<img src={scriptlet_config} width="720" />

## 3. Scriptlet Block 代码

> 支持使用Nodejs和Python处理业务逻辑。支持社区数据库引用，支持完成和AI辅助编程。
