# 📦 Scriptlet 和 Block

## Scriptlet

Scriptlet 是一个核心基础模块。当内置的 Block 无法满足创建工作流的需求时，用户可以基于 Scriptlet 编程进行补充。

<img src={scriptlet} width="720" />

## Block

Block 是由 Scriptlet 转换而来的，也是一个重要的基础模块。当 Scriptlet 的功能具有通用性且开发者希望将其抽象为一个可重复使用的工具时，可以将 Scriptlet 转换为 Block。
