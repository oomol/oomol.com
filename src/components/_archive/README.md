# Archived homepage blocks

Components in this folder are **kept for later composition** (videos, alternate heroes, A/B layouts). They are not deleted from the repo.

- **`HomepageFirstScreenVideoHero.tsx`** — full-width workflow demo video + play overlay used on the homepage hero before the Magic UI–style terminal + beam refresh.

To restore the video in the main fold, import it from `@site/src/components/_archive/HomepageFirstScreenVideoHero` inside `HomepageFirstScreen` (or another page) and place it where you want it to appear.
